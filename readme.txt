Slickgrid - это Widget grid'a на базе framework jquery. Этот grid был выбран в качестве основного компонента отображения табличных данных. Выбор пал именно на этот компонент потому, что он богат на функционал, он постоянно развивается, имеет развитое сообщество, а также просто потому, что остальные не отвечают требованиям:
1. Виртуальный скроллинг
2. Поддержка ajax
3. Поддеркжа celFormatters и celEditors
4. развитый API
